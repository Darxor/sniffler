Sniffler is a tool for collecting information about files in a directory.

How to Use

  1. Go to the Scan page.
  2. Enter the path of the directory you want to scan.
  3. Click the "Start Scan" button to begin the scan.
  4. View the scan results in the scan history section.
  5. You can set a scan as active to access it on other pages.
